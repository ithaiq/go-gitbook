# Golang光速实战
> 此开源图书由[thaiq](https://github.com/ithaiq)原创，创作不易转载请注明出处

* 创作此书的目的
    * 市面上写go语法的很多书，但实战案例的书很少寥寥无几。伴随docker+k8s云原生的潮流让go的生态越来越好，微服务整套架构和K8S部署实战也是每一位核心开发必须掌握的。
    * 我希望Gopher能从实战入门，毕竟go语法简单入门容易但是写好需要不少功力
    * 本书面向读者是懂go基础但希望进阶的偏中上的读者针对人群是偏**运维开发**
    * 本书是自己学习参考一些开源库的实现和自己工作中的思考总结完成，有问题请及时提issues
    * 我非大厂出身也不是牛人文笔不是很好但我希望我写的能对你有所帮助就足够了
    * 星星之火可以燎原，本书永远开源免费

* 后期更新方向(不会按照顺序更新，进度大概每周2、3篇)
    * 第一期：golang基础实战
    * 第二期：go面试准备
    * 第三期：golang微服务（gateway、服务注册、服务治理、链路追踪、grpc、gin等）
    * 第四期：k8s技术栈（入门、二开、servicemesh）
